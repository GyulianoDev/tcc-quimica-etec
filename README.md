# Quimica

Esse projeto e uma tela inicial para uma plataforma de estudos de Quimica.
A ideia e ter um visual mais divertido, com cara de jogo antigo/neon, mas ainda simples de usar.

Por enquanto, a pagina tem:

- tela de login;
- aba para criar conta;
- recuperacao de senha;
- onboarding depois do cadastro;
- escolha de avatar;
- campos de nome completo, nascimento, CPF e tipo de matricula;
- layout responsivo para celular e computador.

## Obs

Ainda e uma tela front-end, entao os formularios nao salvam dados de verdade.
Eles estao prontos visualmente para depois conectar com um sistema de login ou banco de dados.
